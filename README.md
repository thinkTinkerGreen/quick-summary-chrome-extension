# quick-summary-chrome-extension
A chrome extension that quickly summarises the content of the pages behind the links on a page

Watch the demo of this Quick Summary - Chrome extension 
https://www.youtube.com/watch?v=M7uty3OvceM

This is a simple chrome extension that will help you summarise the contents of the page before navigating to it in about 100 to 150 words.
It will save a lot of time if one is going through a page will a lot of links like news.

It uses an LLM model like google gemini or claude or gpt to extract the summary of a page when it is pages the hyperlink of that page. The favoured LLM model can be selected from the drop-down menu of the options or settings page.

It is a simple overlay and intuitive UI design inspired from a music player to traverse through the various links on a page as demonstrated in the video (link mentioned above) for Hacker news.
